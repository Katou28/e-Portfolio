01

Shell Sort Algorithm

This project compares the performance of Shell Sort in sequential and parallel implementations, with the parallel version leveraging OpenMP for improved efficiency. The program evaluates both methods in a worst-case scenario, using reverse-sorted input arrays, and measures execution time with high-resolution clocks. The results highlight the benefits of parallelism in sorting algorithms, showcasing potential speedup and scalability improvements.

02

Wifi Signal Visualizer

The project "WIFI Signal Synchronizer" provides a system for scanning and sharing Wi-Fi signal strength data between a client and server using socket communication. The client application identifies nearby networks, calculates their signal strength in percentage, and sends the data to the server. The server organizes and displays the received data in a user-friendly interface, facilitating efficient Wi-Fi monitoring and synchronization.

03

DISC Personality Test

The project "DiSC Personality Test" is an interactive program that evaluates an individual's personality traits across four categories: Dominance, Influence, Steadiness, and Compliance. It presents a series of questions for each trait, gathers user responses, and calculates scores to determine the most dominant personality quadrant. The program is implemented in assembly language, featuring a user-friendly flow for answering questions and viewing results.
